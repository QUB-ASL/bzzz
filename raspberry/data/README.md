# Data files

CSV files with the following columns:

- `Time-stamp`
- `T_ref`
- `yaw` (rad)
- `pitch`
- `roll`
- `ToF measurement` (mm)
- `accX`, `accY`, `accZ`, 
- `Ref alti`
- `RC data` (string)
- `mot_FL`
- `mot_FR`
- `mot_BL`
- `mot_BR`
- `KF alti est`
- `KF vel est`
- `KF alpha est`
- `KF beta est`
